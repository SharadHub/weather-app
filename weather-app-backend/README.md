# Weather App Backend

A simple Node.js command-line tool that fetches current weather data for any city.

## Features

- Real-time weather data from OpenWeatherMap API
- Interactive city input via terminal
- Displays temperature (°C) and weather conditions

## Requirements

- Node.js (ES modules support)
- npm

## Installation

```bash
npm install
```

## Usage

```bash
node weather.js
```

Enter a city name when prompted. The app will display:
- City name
- Current temperature
- Weather description

## Dependencies

- axios: HTTP client for API requests

## Tech Stack

Node.js, ES Modules, OpenWeatherMap API
